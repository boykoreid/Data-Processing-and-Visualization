# Data-Processing-and-Visualization
This project combines basic data processing, machine learning and visualization techniques to analyze and present employee data.

## Features
#### Data Visualization
- Implemented pie and bar charts using matplotlib
- Visually compared employees who quit vs employees who didn't

#### Machine Learning
- Built a decision tree model using scikit-learn
- Predicted employee attrition based on training data

#### Data Structures (Carousel)
- Designed a Carousel system to display employee records
- Implemented using a doubly linked list
    - Each node represents an employee
    - Supports forward and backward traversal

#### Data Storage
- Imported data from three csv files
- Stored data in a dictionary
- Created a class to clean data
    - Found missing data in categorical columns and removed the row it corresponded with
    - Replaced missing data in numerical columns with the average from that column
- Created a class to help access data in the dictionary

## Technologies Used
- IPython
- matplotlib
- scikit-learn

## Key Learning Outcomes
- Gained experience with data cleaning and preprocessing
- Learned how to visualize data effectively
- Applied machine learning techniques using decision trees
- Strengthened understanding of data structures through practical implementation
